## Format Specifiers

This isn't very useful besides making notes


| Specifier | Description      | Notes                 |
| --------- | ---------------- | --------------------- |
| C         | Calculation      | -                     |
| D         | Definition       | RPGIV Only            |
| E         | File Extension   | After F               |
| F         | File Description | -                     |
| H         | Header (control) | -                     |
| I         | Input            | -                     |
| L         | Line Counter     | After E or F(If no E) |
| O         | Output           | -                     |
| P         | Subprocedure     | RPGIV Only            |
